\section{Введение}

Целью проекта является создание программного продукта, нацеленного
на графическое отображение графов путём отображения содержимого .dot-файлов,
с возможностью последующего редактирования и сохранения.


\section{Основания для разработки}

Основанием для разработки является учебный план кафедры ФТК на 10-й
семестр, утверждённый заведующим кафедрой.


\section{Назначение разработки}

Данный программный продукт является одним из этапов обучения по дисциплине
``Проектирование Программного Обеспечения'', позволяющий освоить
основные навыки разработки и сопровождения программного обеспечения.
Так как работа выполняется в группах, появляется возможность проверить
свою подготовленность для работы в группе.


\section{Требования к программе или программному изделию}


\subsection{Требования к функциональным характеристикам.}

Разрабатываемый программный продукт должен позволять:
\begin{enumerate}
\item Открывать файлы в формате .dot
\item Менять расположение вершин графа по различным существующим алгоритмам
\item Делать масштабирование и навигацию по графу
\item Выделение некоторых подграфов, таких, как единые пути, связные компоненты,
замкнутые кольца.
\item Возможность сворачивания/разворачивания текста внутри вершины
\item Иметь доступный и простой пользовательский интерфейс
\item Иметь гибкую систему настроек
\end{enumerate}

\subsection{Требования к надёжности}

Надёжность системы в целом зависит от используемой операционной системы.


\subsection{Ограничения программного продукта}

Максимальная позиция элемента графа по оси Х или У не должна превышать
значения 2\textasciicircum{}31. 


\subsection{Условия эксплуатации}

Стандартные условия эксплуатации программных продуктов. Специальной
подготовки для использования программного продукта не требуется.


\subsection{Требования к составу и параметрам технических средств}

Для нормальной работы программного продукта, необходимо:
\begin{enumerate}
\item Установленная операционная система семейства Windows XP (SP3 (32-разрядная
версия); Windows XP SP2 (64-разрядная версия), Windows Vista SP2,
Windows 7, Windows 8, Windows Server 2008, Windows Server 2012 (64-разрядная
версия), MacOS X 10.7.3 и выше, Lunux: Oracle Linux 5.5+, Red Hat
Enterprise Linux 5.5+, Ubuntu Linux 10.04 и выше, Suse Linux Enterprise
Server 10 SP2, 11.x 
\item Оперативная память не менее 128МБ
\item Жесткий диск не менее 150МБ
\item Установленное программное обеспечение Java\texttrademark{} Virtual
Machine v7 и выше
\item Наличие видеоадаптера для отображения графического интерфейса
\item Наличие манипулятора типа ``мышь''
\item Наличие клавиатуры
\end{enumerate}

\subsection{Требования к информационной совместимости}

Разрабатываемый программный продукт должен работать под управлением
операционной системы , описанной в П.4.4. Для запуска программного
продукта необходимо наличие установленной виртуальной машины Java\texttrademark{}
Virtual Machine v7 и выше.\\
В качестве средства сборки требуется использовать Maven.\\
В качестве средства для создания графов может быть использован любой
программный продукт, способный сохранять граф в формате .dot.
